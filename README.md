  Best practice:

  1. Keep a clean code-base.py (never edit this).
  2. For each task, copy it to ex1.py, ex2.py, etc.
  3. Modify only what the exercise asks.
  4. Reuse helper functions you already built.

  About internet: yes, use it, but with rules.

  1. Use docs/tutorials only when blocked.
  2. Search by problem, not by full solution.
  3. After reading, close browser and implement from memory.
  4. If your teacher forbids internet during evaluation, practice some sessions offline too.

  Goal: understand the loop and logic, not memorize every command.
